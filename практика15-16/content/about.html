Описание

MEMO — минималистичное приложение для управления задачами, работающее полностью в браузере. Данные хранятся локально, приложение доступно без интернета.

  • App Shell архитектура — мгновенная загрузка интерфейса
  • Service Worker — кэширование и работа офлайн
  • Cache API — две стратегии: Cache First + Network First
  • localStorage — сохранение задач на устройстве
  • Web App Manifest — установка как нативное приложение
  • HTTPS — обязательное условие для Service Worker
